🎬 Movie Match
Swipe. Match. Watch.
Movie Match is a sleek React Native Android application designed to end the "what should we watch tonight?" struggle. Inspired by the intuitive "swipe" mechanics, it helps users discover films based on their preferences in a fun, interactive way.



✨ Key Features
Tinder-style Discovery: Swipe right to "Match" (Save) a movie, swipe left to skip.
Smart Filtering: Filter by genre, release year, and minimum rating (IMDb/TMDB).
Rich Movie Details: Access trailers, synopses, cast information, and high-quality posters.
Personal Watchlist: All your "Matches" are saved in a dedicated list for quick access later.
Offline Mode: Browse your saved matches even without an internet connection (via local storage).
Smooth Animations: Powered by Reanimated 2 for a fluid, high-performance UI experience.
🚀 Tech Stack
Framework: React Native
Navigation: React Navigation (Stack & Tabs)
Animations: Reanimated + React Native Gesture Handler
Data Source: TMDB API
State Management: Redux Toolkit / Context API
Storage: AsyncStorage
